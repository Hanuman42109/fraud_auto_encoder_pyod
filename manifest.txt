Project: Credit Card Fraud Detection with AutoEncoder (PyOD)
Dataset: Kaggle Credit Card Fraud Detection (creditcard.csv)
Libraries: pyod, pandas, numpy, scikit-learn, matplotlib, tqdm, torch, torchvision, torchaudio
Python Version: 3.9+
IDE: Visual Studio Code
Author: Hanuman Sai Chanukya Srinivas Chilamkuri
Date: 2025-08-15

Instructions:
1) Create a Python venv and install requirements:
   python -m venv .env
   # Windows PowerShell:
   .venv\Scripts\Activate.ps1
   # Mac/Linux:
   source .venv/bin/activate
   pip install -r requirements.txt

2) Download Kaggle dataset and place file at ./data/creditcard.csv

3) Run:
   python src/fraud_autoencoder_pyod.py --data ./data/creditcard.csv --outputs ./outputs

Artifacts saved under ./outputs:
- metrics.txt
- confusion_matrix.png
- score_histogram.png
